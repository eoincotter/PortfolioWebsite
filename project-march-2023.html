March 2023 | Create a Portfolio Website

The aim of this project was to develop a portfolio website to document and present my self-guided analytics projects. I initially planned to use a website builder such as WordPress or Wix but, having tried them out and researched costs, instead I decided to develop the site from scratch using HTML and CSS. I’m not a web developer but I do have some background in web development through a university module and some self-guided study. While this project isn’t strictly data or behavioural science, I have decided to include it as part of my portfolio because it demonstrates some transferrable skills, and web development knowledge is certainly useful to have in any data role. Below I outline the steps I took to complete the project.

Contents

Add Home page content with HTML

As a starting point I wanted the website to contain two main pages: Home and Projects. The Home page should provide a brief introduction to myself and provide links to my CV and professional social media pages. The Projects page should provide a list of my projects with a brief description and a link to the full project report. The first step to achieve this was to add the basic content (text, images) using HTML. Using Notepad++ as my code editor, I first created index.html as the homepage, starting with empty <head> and <body> elements. 

			
 <html>
    <head>
    </head>
    <body>
    </body>
 </html>

I then added a webpage title to <head>, which appears on search results and on the browser tab when the page is opened. 

			
 <head>
    <title> Eoin Cotter </title>
 </head>

Next I added a header into <body> which will act as a navigation bar with links to each of two main pages. 

			
 <body>
    <header>
        <a href = "index.html" class = "no-underline" class="current-page"> <p> HOME </p> </a>
        <a href = "projects.html" class = "no-underline"> <p> PROJECTS </p> </a> 
    </header>
 </body>

I then added a <section> element to the <body> which contains the main content for the page. For the Home page, the main content includes a heading with my name, a paragraph with information about me, and image links to my CV and professional social media pages. 

			
 <section>
    <h1> Eoin Cotter | Data & Behavioural Science </h1>
    <h4> Melbourne, VIC </h4>
    <p> I hold a Bachelor of Commerce, a Masters in Business Analytics, and a Higher Diploma in Psychology. I’m interested in using a scientific, 
	data-driven approach to gain insights into the world around me, especially human behaviour. This website documents my efforts to become better at doing so through self-guided projects.</p>
    <a href="https://www.linkedin.com/in/eoinwcotter/"> <img src="LinkedinIcon.png" alt="LinedIn Icon" width="30px" height="30px"> </a>
    <a href="https://github.com/eoincotter"> <img src="GithubIcon.png" alt="GitHub Icon" width="30px" height="30px"> </a>
    <a href="cv.pdf"> <img src="CV.png" alt="CV Icon" width="30px" height="30px"> </a>
 </section>

And that completed the content for the home page. The next task was to add styling with CSS.

Add styling with CSS

To begin styling I created a blank css file, styling.css, and linked it to index.html by adding the below <link> element to <head>. 

			
<link rel="stylesheet" href="styling.css" type = "text/css">
